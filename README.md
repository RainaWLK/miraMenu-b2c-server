# miraMenu-b2c-server
Serverless B2C backend for MiraMenu

Use Lambda + API gateway

### Init
```
npm install
```

### Local Development
```
npm start
```

test
```
npm test
```

### Deploy to AWS (first time)
```
npm run create
```

### Deploy
```
npm run deploy
```

release to production
```
npm run release
```

### Destroy
```
npm run destroy
```
